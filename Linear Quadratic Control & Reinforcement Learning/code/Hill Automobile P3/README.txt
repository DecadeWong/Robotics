Here are two directory for problem 3. One is for monte carlo method, and the other one is for time temporal difference method.

To run  monte carlo method, just open and run MonteCarlo.py in MCfinal_version folder. You can change the total episodes in line 91, and you can also comment out line 99.


To run  temporal difference method, just open and run TemporalDifference.py in TDfinal_version folder. You can change the total episodes in line 76, and you can also comment out line 83.
